### Test Repository instructions

#### Test tool & command
Tests will be run using `maven clean test`.

#### Project structure
Make sure that the package structure of your test classes is equivalent to the package structure of your base and solution repository.
Otherwise during the test run maven will not be able to find the imported classes in your test files.

### Forking and copying
It is recommended to fork this repository for new exercises to make new features and fixes available to all exercises. You can also copy the test files to your exercise repository, but then you will have to maintain them separately for each exercise.
